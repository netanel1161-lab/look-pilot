מדריך להרצת סקריפט עיבוד נתוני הרכב (Process Gov Data)

סקריפט זה לוקח קובץ נתונים ממשלתי (CSV או Excel) וממיר אותו לקובץ JSON נקי ומסודר שמתאים למערכת שלך.

דרישות קדם (Prerequisites):

ודא שיש לך Python מותקן על המחשב.
לבדיקה, פתח שורת פקודה (CMD או Terminal) וכתוב: python --version

ודא שמותקנות הספריות הבאות:
פתח שורת פקודה וכתוב:
pip install pandas openpyxl

הוראות הפעלה:

שמירת הסקריפט:

ודא שהקובץ 'process_gov_data.py' נמצא בתיקייה שבה תרצה לעבוד.

הכנת קובץ הנתונים:

הורד את קובץ המאגר הממשלתי (לרוב קובץ CSV גדול).

שים את הקובץ באותה תיקייה בדיוק שבה נמצא הסקריפט.

ודא שסיומת הקובץ היא .csv או .xlsx.

הרצת הסקריפט:

פתח את הטרמינל (שורת הפקודה) ונווט לתיקייה שבה שמרת את הקבצים.
(למשל: cd C:\Users\YourName\Desktop\MyProject)

כתוב את הפקודה הבאה ולחץ Enter:
python process_gov_data.py

מה יקרה כעת?

הסקריפט יחפש אוטומטית קובץ CSV בתיקייה.

הוא יטען את הנתונים (זה עשוי לקחת כמה שניות אם הקובץ גדול).

הוא ינקה כפילויות, ימיין את היצרנים והמודלים לפי סדר א'-ב'.

בסיום, ייווצר בתיקייה קובץ חדש בשם: 'vehicle_db_clean.json'.

שימוש בתוצאה:

את הקובץ החדש שנוצר ('vehicle_db_clean.json') עליך להעתיק לתיקיית ה-data בפרויקט ה-Web שלך, או לטעון אותו דרך הכפתור "טען מאגר רכבים" בכלי שבנינו.

בהצלחה!
